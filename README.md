# Cloud-Based Bus Pass Management System

A cloud-based web application developed using Flask that enables users to book, manage, and digitally access bus passes with secure session handling and administrative controls.

---

## Key Features

- User Registration and Login
- Digital Bus Pass Generation with Unique Pass ID
- Route and Duration Selection
- Automatic Price Calculation
- Booking History Tracking
- Pass Cancellation System
- Admin Dashboard with Revenue and Booking Analytics
- Secure Session Management
- SQL Injection Protection using Parameterized Queries
